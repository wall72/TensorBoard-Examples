# Notebook execution results

이 폴더는 실행 완료된 노트북 HTML 결과물을 저장하기 위한 위치입니다.

## 생성 방법

```bash
python scripts/execute_notebooks.py
```

실행이 완료되면 아래 파일들이 생성됩니다.

- `results/01-softmax.html`
- `results/02-nn.html`
- `results/03-nn-xavier.html`
- `results/04-nn-xavier-dropout.html`
- `results/05-cnn.html`

> 참고: TensorFlow 2.15 설치가 필요합니다.
